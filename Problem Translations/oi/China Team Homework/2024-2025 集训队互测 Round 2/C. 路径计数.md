# C. 경로 개수 세기
C. 路径计数

## 문제 설명
$n$ 행 $m$ 열의 그리드가 있습니다. 그리드 위에는 총 $(n+1) \times (m+1)$ 개의 격자점이 존재하며, 이 중 $x$ 행 $y$ 열의 격자점은 순서쌍 $(x, y)$ 로 나타냅니다 (격자점의 행과 열은 모두 0부터 번호가 매겨집니다).

초기에 그리드에는 간선이 없습니다. 이제 총 $(3m+1)n$ 개의 방향 간선을 차례대로 추가합니다:
* $0 \le i \le n-1, 0 \le j \le m-1$ 에 대해, $(i, j)$ 에서 $(i+1, j+1)$ 로 향하는 본질적으로 서로 다른(essentially different) 방향 간선을 $A_j$ 개 추가합니다.
* $0 \le i \le n-1, 0 \le j \le m$ 에 대해, $(i, j)$ 에서 $(i+1, j)$ 로 향하는 본질적으로 서로 다른 방향 간선을 $B_i + C_j$ 개 추가합니다.
* $0 \le i \le n-1, 1 \le j \le m$ 에 대해, $(i, j)$ 에서 $(i+1, j-1)$ 로 향하는 본질적으로 서로 다른 방향 간선을 $D_j$ 개 추가합니다.

이제 $0 \le x \le n, 0 \le y \le m$ 을 만족하는 정수 $x, y$ 에 대해, $W(x, y)$ 를 $(0, 0)$ 에서 $(x, y)$ 로 가는 본질적으로 서로 다른 경로의 개수로 정의합니다. (경로의 개수가 유한함은 쉽게 증명할 수 있습니다.)
당신은 다음 수식의 결과를 구해야 합니다:

$$
\sum_{i=0}^{n} \sum_{j=0}^{m} W(i, j) E_i F_j \pmod p
$$

## 입력 형식
첫 번째 줄에 네 개의 양의 정수 $c, n, m, p$ 가 입력됩니다. 첫 번째 수는 서브태스크 번호를 나타내며 (특별히 예제에서의 $c$ 는 해당 예제가 $c$ 번째 서브태스크의 제한 조건을 만족함을 의미합니다), 두 번째와 세 번째 수는 그리드의 크기, 네 번째 수는 정답을 나눌 모듈로(modulo) 값을 의미합니다.
두 번째 줄에는 $m$ 개의 수가 입력되며, $i$ 번째 수는 $A_{i-1}$ 의 값을 나타냅니다.
세 번째 줄에는 $n$ 개의 수가 입력되며, $i$ 번째 수는 $B_{i-1}$ 의 값을 나타냅니다.
네 번째 줄에는 $m+1$ 개의 수가 입력되며, $i$ 번째 수는 $C_{i-1}$ 의 값을 나타냅니다.
다섯 번째 줄에는 $m$ 개의 수가 입력되며, $i$ 번째 수는 $D_i$ 의 값을 나타냅니다.
여섯 번째 줄에는 $n+1$ 개의 수가 입력되며, $i$ 번째 수는 $E_{i-1}$ 의 값을 나타냅니다.
마지막 줄에는 $m+1$ 개의 수가 입력되며, $i$ 번째 수는 $F_{i-1}$ 의 값을 나타냅니다.

## 출력 형식
수식 $\sum_{i=0}^{n} \sum_{j=0}^{m} W(i, j) E_i F_j \pmod p$ 의 결과를 나타내는 정수 하나를 한 줄에 출력합니다.

## 데이터 범위 및 서브태스크
모든 데이터에 대해 $1 \le n, m \le 2 \times 10^5$ , $1 \le p \le 10^9$ , $0 \le A_i, B_i, C_i, D_i, E_i, F_i < p$ 를 보장합니다. $p$ 가 반드시 소수인 것은 보장하지 않으나, $p \neq 998244353$ 인 데이터에 대해서는 $1 \le n, m \le 10^5$ 를 만족합니다.
